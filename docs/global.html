date(date, options) → {object}

Returns date object from timestamp (UTC). Or fix received object to default tempus date object.
Parameters:
Name Type Description
date number | object Date as timestamp or object.
options object | undefined Options object.
Properties
Name Type Description
week bool Add to response number of week. Default is false.
dayOfWeek bool Add to response day of week. Default is false.
Source:
  • tempus.js, line 257
Returns:
date - Tempus date object.
Type
object
Examples
// returns {"year":2013,"month":11,"day":5,"hours":16,"minutes":1,"seconds":53}
tempus.date(1383667313);
// returns {"year":2013,"month":11,"day":5,"hours":0,"minutes":0,"seconds":0}
tempus.date(1383609600);
// returns {"year":1970,"month":1,"day":1,"hours":0,"minutes":0,"seconds":0}
tempus.date({});
// returns {"year":2013,"month":10,"day":5,"hours":0,"minutes":0,"seconds":0}
tempus.date({year:2013, month:10, day:5});

time(date, format) → {number}

Returns current timestamp (UTC) in seconds. If "date" parameter is not undefined, timestamp was received from this.
Parameters:
Name Type Description
date object | string Date as object {year: 2013, month: 1, day: 2, hours: 15, minutes: 10, seconds: 5} or string (any formatted date, see examples)
format string | undefined Date format as string (see formats doc) or undefined for autodetect format.
Source:
  • tempus.js, line 220
Returns:
UTC in seconds.
Type
number
Examples
// get current UTC
// returns 1384252977
tempus.time();
// parse date and get UTC
// returns 1381795200
tempus.time('15.10.2013', '%d.%m.%Y');
// returns 1381795200
tempus.time('15.10.2013');
// returns 1383609600
tempus.time({year: 2013, month: 11, day: 5});
// returns 1363046400
tempus.time('2013-03-12', '%Y-%m-%d');
// returns 1363360860
tempus.time('2013-03-15 15:21', '%Y-%m-%d %H:%M');
